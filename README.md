# Ethan.xia Skills

A collection of custom skills for extending functionality.

## Skills

### 🎂 Birthday Reminder
Manage birthdays with natural language. Store birthdays, get upcoming reminders, and calculate ages.

**Features:**
- Natural language birthday storage
- Upcoming birthday notifications
- Age calculation

**Location:** `birthday-reminder/`

### 📧 Gmail
Gmail API integration with managed OAuth authentication. Read, send, and manage emails, threads, labels, and drafts.

**Features:**
- OAuth authentication via Maton API
- Read and send emails
- Manage threads and labels
- Draft management
- Search and filter

**Location:** `gmail/`
