# 小裕的自定义skill集合。

## 技能列表

### 🎂 生日提醒器
使用自然语言管理生日。存储生日信息、获取即将到来的提醒并计算年龄。

**功能：**
- 自然语言生日存储
- 即将到来的生日通知
- 年龄计算

**位置：** `birthday-reminder/`

### 📧 Gmail
带有托管 OAuth 认证的 Gmail API 集成。读取、发送和管理电子邮件、对话、标签和草稿。

**功能：**
- 通过 Maton API 进行 OAuth 认证
- 读取和发送邮件
- 管理对话和标签
- 草稿管理
- 搜索和过滤

**位置：** `gmail/`
